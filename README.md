## 스터디 방법 (아직 확정이 안됐어요!)

크게 두 가지 방법으로 진행됩니다.

### 공동번역
- 스터디 멤버는 자기가 읽을 분량과 글을 선택합니다. 
- 맡은 부분을 번역해서 깃헙에 PR 보냅니다.
- 읽다가 이해안되거나 의문나는 부분이 있으면 이슈게시판에 질문으로 적어 둡니다.
